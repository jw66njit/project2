Domain Names and ICANN

ICANN is a non-profit organization founded in 1998 that hosts the Domain Name System (DNS) and Internet Protocols(IP). It is a way to organize IP addresses and domain names to make it easier for users to access the internet. A domain name is given to an IP address so that it is easier for the user to identify the website. A domain name is split into two categories, In www.facebook.com, facebook would be the domain name and .com would be the top-level domain(TLD). Each TLD (.com,.net,.org,etc) has its own set of servers. When a user types in a web address it sends a request to a DNS server that looks for the IP address, and then sends another request to the TLD server that will look for the domain name.

ex. User types in the domain name facebook.com. The user sends a request to a DNS server for the IP address of facebook, and that DNS server will send a request to a TLD server (.com) to find facebook.com and its IP address to send back to the user. The IP address will be saved in the DNs server so the next time the user request access the loading time will be faster.
